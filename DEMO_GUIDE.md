# 🎯 คู่มือการ Demo สำหรับการพิทช์

## 📱 Flow การ Demo (5-7 นาที)

### **1. เปิดหน้าแรก (1 นาที)**

```
URL: http://localhost:3000
```

**สิ่งที่ต้องบอก:**
> "นี่คือแพลตฟอร์ม IAMROOT AI - เหมือน Google สำหรับโปรโมชั่น แต่ทำงานแบบ Mobile-first เพราะกลุ่มเป้าหมายเราใช้มือถือ"

**จุดเด่นที่ต้องชี้:**
- Header มี Search bar เด่นที่สุด
- แถบสถิติ "35,000+ โปรโมชั่น" และ "ตรวจสอบโดย PIM"
- โปรโมชั่นที่มี Badge "แนะนำ" (Sponsored) อยู่อันดับ 1-3
- โปรโมชั่น Verified จาก 7-11, Lotus, Makro

---

### **2. ทดสอบ Search Engine (1.5 นาที)**

**ขั้นตอน:**
1. คลิกที่ Search bar
2. พิมพ์ **"Lotus"**
3. ดูผลลัพธ์ที่โผล่มาพร้อม Auto-suggest

**สิ่งที่ต้องบอก:**
> "ระบบ SEO ของเรามี 3 Tiers:
> - **Tier 1 (Sponsored):** ร้านค้าจ่ายเงินซื้อ SEO จะขึ้นอันดับ 1-3
> - **Tier 2 (Verified):** ข้อมูลจากเครือ CP ALL - นี่คือ Unfair Advantage ของเรา
> - **Tier 3 (General):** ร้านค้าลงฟรี เรียงตาม search volume"

**ลองค้นหาเพิ่ม:**
- **"กาแฟ"** → จะเห็นโปรโมชั่นเครื่องดื่ม
- **"ข้าว"** → จะเห็นทั้ง 7-11, Lotus, Makro

---

### **3. คลิกดูรายละเอียดโปรโมชั่น (1.5 นาที)**

**ขั้นตอน:**
1. เลือกโปรโมชั่นที่มี Badge "Verified" (เช่น จาก 7-Eleven)
2. คลิกเข้าไปดู

**สิ่งที่ต้องบอก:**
> "ดูครับ มี **Verification Banner** บอกว่าตรวจสอบโดย PIM Innovation Center พร้อมวันที่ตรวจสอบล่าสุด - สิ่งนี้สร้างความน่าเชื่อถือ"

**จุดเด่นที่ต้องชี้:**
- Verification Banner (สีเขียว)
- ราคาเดิม vs ราคาพิเศษ
- สถิติ Views, Saves, Clicks (ด้านล่าง)
- ปุ่ม "รายงานข้อผิดพลาด" → **Crowdsourcing**

**ทดสอบ Report:**
1. คลิก "รายงานข้อผิดพลาด"
2. พิมพ์เหตุผล (เช่น "ราคาไม่ถูกต้อง")
3. แสดงให้เห็นว่าระบบรับเรื่อง

> "เราใช้ Community Crowdsourcing ให้คนช่วยตรวจสอบข้อมูล - ระบบจะตรวจสอบภายใน 24 ชั่วโมง"

---

### **4. เปิด Merchant Dashboard (2-3 นาที)** ⭐ **จุดขายหลัก**

```
URL: http://localhost:3000/merchant/dashboard
```

**สิ่งที่ต้องบอก:**
> "นี่คือ **ไฮไลท์ของเรา** - Merchant Dashboard ที่จะทำให้เราต่างจากปันโปรชัดเจน"

#### **A. Data Insights Section**

**จุดเด่นที่ต้องชี้:**
- **สถิติการค้นหา:** แสดงจำนวน searches ในเดือนนี้
- **คำค้นหายอดนิยม:** แสดง Top 10 Keywords ในย่านที่เลือก
- **Insight Box (สีฟ้า):**

> "ดูครับ มันบอกว่า **'คนในย่านอารีย์กำลังค้นหา นมโปรตีน เพิ่มขึ้น 50%'**
> 
> นี่คือข้อมูลที่ **เจ้าของร้านต้องการ** เพราะจะช่วยวางแผนโปรโมชั่นได้ตรงใจลูกค้า
> 
> และนี่คือสิ่งที่ **ปันโปรทำไม่ได้** เพราะเราเชื่อมกับ CP ALL Ecosystem"

**ทดลอง:**
- เปลี่ยนย่าน (Dropdown) → ข้อมูลจะเปลี่ยนตาม

#### **B. SEO Manager Section**

**จุดเด่นที่ต้องชี้:**
- แพ็กเกจ 3 แบบ: Basic (299฿), Pro (899฿), Premium (2,499฿)
- คลิกเลือกแพ็กเกจ → แสดง Banner ชำระเงิน

**สิ่งที่ต้องบอก:**
> "นี่คือ **Business Model** ของเรา:
> 
> 1. **ร้านค้าลงโปรโมชั่นฟรี** (ไม่เสียค่าใช้จ่าย)
> 2. แต่ถ้าอยากให้โปรโมชั่น **ขึ้นอันดับต้น** ต้องซื้อ SEO Package
> 3. จ่าย 299 บาท ได้อันดับ 4-10 / จ่าย 899 บาท ได้อันดับ 1-3 / จ่าย 2,499 บาท ได้อันดับ 1 การันตี
> 
> **ชัดเจน โปร่งใส และทำกำไรได้จริง**"

#### **C. Activity Log**

**จุดเด่นที่ต้องชี้:**
- แสดงสถิติการ View, Save, Click แบบเรียลไทม์
- ร้านค้าจะรู้ว่าโปรโมชั่นไหนได้รับความสนใจมากที่สุด

---

### **5. กลับไปหน้าแรก + สรุป (1 นาที)**

**สิ่งที่ต้องบอก:**
> "สรุปครับ IAMROOT AI แตกต่างจากปันโปร 4 จุดหลัก:
> 
> 1. **Unfair Advantage:** เรามีข้อมูลจริงจากเครือ CP ALL (7-11, Lotus, Makro) 10,000+ สาขา
> 2. **Data Monetization:** เราขาย Data Insights ให้ร้านค้า ไม่ใช่แค่พื้นที่โฆษณา
> 3. **SEO Ranking แบบโปร่งใส:** จ่ายเท่าไหร่ ได้อันดับชัดเจน
> 4. **Verification System:** สร้างความน่าเชื่อถือ + Community Crowdsourcing
> 
> เราไม่ได้แข่งกับปันโปร เราสร้างตลาดใหม่ที่มี **Data** เป็นศูนย์กลาง"

1. **พูดช้าๆ ชัดเจน** - ให้กรรมการตามทัน
2. **ชี้หน้าจอ** - เน้นจุดสำคัญ
3. **เล่าเป็นเรื่องราว** - "ลองจินตนาการว่าคุณเป็นเจ้าของร้านกาแฟ..."
4. **เน้น Unfair Advantage** - ทุกครั้งที่มีโอกาส
5. **ตอบคำถามด้วยข้อมูล** - อ้างอิงจาก Dashboard
