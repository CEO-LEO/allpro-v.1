# 🎯 จุดขายหลักสำหรับการพิทช์

## 1️⃣ Problem (ปัญหาที่แก้)

### ปัญหาของผู้บริโภค:
- หาโปรโมชั่นยาก ต้องเปิดหลายแอป
- ข้อมูลไม่ครบ ไม่ตรงกับความเป็นจริง
- โปรโมชั่นหมดอายุแล้วยังแสดงอยู่

### ปัญหาของร้านค้า:
- ไม่รู้ว่าลูกค้าต้องการอะไร
- ลงโปรโมชั่นแล้วไม่มีคนเห็น
- แข่งขันกับร้านใหญ่ยาก

---

## 2️⃣ Solution (วิธีแก้ของเรา)

### สำหรับผู้บริโภค:
✅ **One-Stop Search** - ค้นหาโปรโมชั่นทุกร้านในที่เดียว  
✅ **Verified Data** - ข้อมูลตรวจสอบโดย PIM Innovation Center  
✅ **Location-based** - แสดงโปรโมชั่นใกล้เคียง (Future)

### สำหรับร้านค้า:
✅ **Data Insights** - รู้ว่าลูกค้าต้องการอะไร  
✅ **SEO Ranking** - ซื้อ SEO ให้โปรโมชั่นขึ้นอันดับต้น  
✅ **Free Listing** - ลงโปรโมชั่นฟรี ไม่เสียค่าใช้จ่าย

```
ไม่ได้ขายแค่พื้นที่โฆษณา แต่ขาย "ข้อมูล"
```

**ตัวอย่าง:**
> "คนในย่านอารีย์กำลังค้นหา 'นมโปรตีน' เพิ่มขึ้น 50%"

- Fair Play สำหรับทุกคน

### 🔥 #4: Verification System
```
Verified by PIM Innovation Center
+ Community Crowdsourcing
```

**ทำไมสำคัญ?**

## 4️⃣ Business Model (โมเดลธุรกิจ)

### รายได้หลัก 3 แหล่ง:

#### A. SEO Ranking Packages (80% รายได้)
| Package | ราคา/เดือน | เป้าหมาย Year 1 | รายได้/เดือน |
|---------|-----------|-----------------|-------------|
| Basic | 1,196฿ | 500 ร้าน | 598,000฿ |
| Pro | 899฿ | 300 ร้าน | 269,700฿ |
| Premium | 2,499฿ | 200 ร้าน | 499,800฿ |
| **รวม** | | **1,000 ร้าน** | **1,367,500฿** |

#### B. Data Insights Premium (15% รายได้)
- ขาย Data Report ขั้นสูงให้ร้านค้า
- ราคา: 999฿/เดือน
- เป้าหมาย: 200 ร้าน
- รายได้: 199,800฿/เดือน

#### C. Sponsored Campaign (5% รายได้)
- แคมเปญพิเศษ (เช่น "เทศกาล")
- ราคา: 5,000-50,000฿/แคมเปญ

### รายได้รวม Year 1:
```
SEO: 1,367,500฿/เดือน x 12 = 16,410,000฿
Data: 199,800฿/เดือน x 12 = 2,397,600฿
Sponsored: ~1,000,000฿/ปี

= ~20,000,000฿/ปี (Year 1)
```

## 6️⃣ Go-to-Market Strategy (กลยุทธ์)

### Phase 1: Launch (เดือน 1-3)
1. **Partner กับ CP ALL** - ดึงข้อมูล 7-11, Lotus, Makro
2. **Onboard 100 ร้านค้าแรก** (ฟรี 3 เดือน)
3. **PR + Social Media** - โปรโมทผ่าน FB, TikTok

### Phase 2: Growth (เดือน 4-12)
1. **Scale ไป 1,000 ร้าน**
2. **เปิดตัว SEO Packages**
3. **Data Insights Premium**
4. **Mobile App**

### Phase 3: Expand (Year 2+)
1. **ขยายไปภูมิภาค** (เชียงใหม่, ภูเก็ต, ขอนแก่น)
2. **International** (เวียดนาม, อินโดนีเซีย)
3. **API สำหรับ Third-party**
